# v0.1.0 – Estructura inicial del módulo

**Fecha:** 2026-04-04  
**Tipo:** Initial release

## Cambios incluidos

### Repositorio

- README.md actualizado con descripción completa, estructura y guía de inicio rápido.
- `.gitignore` configurado para Flutter, Android, iOS y Supabase.

### Documentación (`docs/`)

- `docs/index.md` – índice de documentación.
- `docs/DOCUMENTACION_POR_PR.md` – checklist de documentación por PR.
- `docs/MIGRATION_GUIDELINES.md` – convenciones para migraciones SQL.
- `docs/AUDITORIA_STANDARD.md` – estándar de auditoría (preparación futura).
- `docs/supabase-workflow.md` – flujo de trabajo con Supabase CLI.
- `docs/user/guia-administradores.md` – manual de la app interna.
- `docs/user/guia-empleados.md` – manual de la app móvil para empleados.
- `docs/dev/arquitectura.md` – arquitectura técnica del sistema.
- `docs/dev/modelos-datos.md` – esquema de tablas y relaciones.
- `docs/dev/setup.md` – configuración del entorno de desarrollo.

### Base de datos (`supabase/migrations/`)

- `20260404000001_add_columns_empleados.sql` – extensión de la tabla empleados.
- `20260404000002_create_feriados.sql` – tabla de feriados.
- `20260404000003_create_turnos_personal.sql` – tabla principal de turnos.
- `20260404000004_create_feriado_trabajado.sql` – feriados normales trabajados.
- `20260404000005_create_feriado_especial_trabajado.sql` – feriados especiales trabajados.
- `20260404000006_create_especial_rotacion.sql` – rotación anual de feriados especiales.

### Flutter – App admin (`apps/admin/`)

- Estructura base del proyecto Flutter.
- Modelos: `Turno`, `Empleado`, `Feriado`, `FeriadoTrabajado`, `EspecialRotacion`.
- Servicios: `SupabaseService`, `TurnosService`, `FeriadosService`, `EmpleadosService`.
- Providers: `TurnosProvider`, `FeriadosProvider`, `EmpleadosProvider`.
- Pantallas: `TableroSemanalScreen`, `EditarTurnoScreen`, `ResumenMensualScreen`, `FeriadosScreen`.

### Flutter – APK empleados (`apps/empleados/`)

- Estructura base del proyecto Flutter.
- Modelos: `Turno`, `Empleado`.
- Servicio: `TurnosService`.
- Pantallas: `SemanaTurnosScreen`, `ActividadMensualScreen`.
